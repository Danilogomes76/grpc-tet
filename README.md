> Demo some concepts for Grpc with NodeJS

## Prerequisites

- node >=14.0.0
- [buf](https://github.com/bufbuild/buf)

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev
```

## Applications

  [example](https://github.com/grpc/grpc/blob/master/examples/node/dynamic_codegen/README.md).
